# Product Overview

This workspace contains two main projects being combined to create "Radd" (دستیار راد) - an AI-powered business workspace for Persian-speaking users.

## Components

### Kilocode (kilocode/)
An AI coding assistant VS Code extension (fork of Roo Code). Provides:
- Natural language code generation
- Task automation and verification
- Terminal command execution
- Browser automation
- Multi-model AI support (OpenAI, Anthropic, Ollama, etc.)
- Memory bank for project context
- Checkpointing via shadow Git

### Void Editor (void/)
A VS Code fork designed for AI integration. Provides:
- Desktop application shell
- File explorer and terminal
- Extension system
- React-based AI sidebar UI

## Target Product
A Windows desktop application for business professionals that combines:
- Document organization (folder-based)
- AI agent with full data access
- Q&A with citations
- Research and analysis automation
- Long-running workflow execution
- Persian (Farsi) localization with RTL support

## Key User Personas
- Financial analysts needing document analysis
- Project managers tracking documentation
- Business consultants doing research
